# 📊 Análise de Clientes — Contexto iFood

## 🎯 Objetivo
Analisar o perfil e comportamento dos clientes para identificar 
oportunidades de aumento de faturamento e geração de valor para o negócio.

## 🛠️ Ferramentas
![SQL](https://img.shields.io/badge/SQL-BigQuery-blue)
![Looker Studio](https://img.shields.io/badge/Looker-Studio-orange)

- **SQL (BigQuery)** — tratamento e análise
- **Looker Studio** — dashboard interativo
- **Canva** — apresentação dos resultados

## 📁 Sobre os Dados
Base educacional com informações de perfil de clientes (renda, educação, 
estado civil, filhos), comportamento de consumo e interações com campanhas.

## 🔧 Etapas do Projeto
1. **ETL** — remoção de duplicatas, tratamento de nulos, criação de variáveis derivadas
2. **EDA** — análise descritiva, identificação de padrões
3. **Análise de comportamento** — renda x gasto, filhos x gasto
4. **Geração de insights** — oportunidades de negócio e recomendações

## 💡 Principais Insights
- Clientes de **alta renda** geram a maior parte da receita
- A base de **renda média** representa oportunidade de crescimento
- Clientes **sem filhos** apresentam maior gasto médio

## 🚀 Recomendações de Negócio
| Segmento | Ação Recomendada |
|----------|-----------------|
| Alta renda | Programa de fidelidade premium |
| Renda média | Campanhas para aumento de ticket médio |
| Famílias | Ofertas e combos segmentados |
